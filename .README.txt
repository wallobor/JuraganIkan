Web Game ini dikembangkan dengan Godot Engine 4.4
Web Game ini kemudian di Export menjadi HTML5-based Game dengan framework custom dari Godot.
Web Game ini HARUS dibuka dengan cara "HTML SERVE"
index.html TIDAK bisa dibuka langsung pada browser, melainkan harus dilakukan HTML Serving.

Berikut adalah beberapa cara untuk melakukan testing untuk Web Game ini:

1. Melakukan HTML Serve pada localhost menggunakan Python.
    - Ekstrak ZIP kedalam folder yang diinginkan.
    - Klik kanan pada area kosong folder, lalu jalankan CMD pada direktori tersebut. (Open in Terminal/Command Prompt)
    - Ketik 'python3 -m http.server 8000' pada Terminal (Python 3)
    - Atau ketik 'python -m http.server 8000' jika cara diatas gagal
    - Buka browser dan ketik 'http://localhost:8000/' lalu enter
    - Web Game akan terbuka dan bisa di mainkan.

2. Dengan mengunjungi laman Github kami, 
Untuk mempercepat HTML Serving, kami sudah terlebih dahulu men-deploy Web Game kedalam Github

Web Game sudah ter-deploy dan bisa di mainkan dan dilakukan uji coba secara langsung.
Berikut adalah laman Web Game yang sudah disediakan: https://wallobor.github.io/JuraganIkan/

Untuk memastikan bahwa Web Game yang disubmit sama dengan yang berada di Github kami,
bisa mengunjungi https://github.com/wallobor/JuraganIkan untuk mereview file.


CATATAN PENTING:
HTML Serving manual diatas hanya perlu dilakukan UNTUK membuka Web Game ini dari mesin pribadi.
HTML Serving secara kasarnya adalah menjadikan mesin pribadi seperti komputer/laptop menjadi sebuah Web Server,
yang bisa diakses dengan localhost device tersebut.
HTML Serving sudah otomatis diproses saat .ZIP Web Game ini di unggah dan digunakan dalam Web Server yang menyala.

Di dalam .ZIP Web Game ini akan berisi dengan:
1. index.html
2. Aset penting yang DIBUTUHKAN oleh index.html untuk berkerja dan akan di proses SECARA OTOMATIS.

Saat mengunggah Web Game ini kedalam Web Server, mohon untuk tidak memisahkan atau mengubah tata letak, 
nama file, dan isi konten dari masing-masing file untuk menghindari kesalahan teknis.
